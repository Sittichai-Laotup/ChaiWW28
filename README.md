# Chai · Weekly Engineering Report — WW#28 (06–10 Jul 2026)

An interactive, single-file HTML weekly report for Assembly / Tooling Engineering,
covering the BMS and HSG-HDD programs for Work Week 28.

## View

Open [`index.html`](./index.html) in any modern browser.
Everything is self-contained (inline CSS/JS, no external dependencies).

Because the report is named `index.html`, static hosts (Vercel, Netlify,
GitHub Pages, …) serve it at the site root automatically — no extra config.

> Access code: `chai888`

## Contents

- **BMS** — ONE Drone, NVgotion, Shinzung ASH FORD, CEC, Volkswagen
- **HSG-HDD** — Comet 8233, Dorado 5D 8231, Marlin 10D

## Features

- Blueprint-style light/dark theme (blue & white), toggleable
- Animated KPI counters and a program status donut (On Track / At Risk / Pending)
- July delivery timeline with plan-receive & plan-load milestones
- Filter programs by division (All / BMS / HSG-HDD)
- Live search across part numbers, tooling references and models
- Collapsible program cards with tooling refs, model chips and milestone dates
- Responsive layout, keyboard focus states and reduced-motion support
